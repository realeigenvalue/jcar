# jcar

A car Java maze game using Java Graphics and Java Swing.

![Application Image](jcar.png)

## Building

Prerequisites
- Java JDK
- Gradle Build Tool

```bash
git clone https://github.com/realeigenvalue/jcar.git
cd jcar
gradle build
gradle run
```
